# Recovery Threshold Analysis (MLOps Hands-On)

## Project Overview
This project analyzes customer recovery strategies around a financial threshold using
statistical tests and regression modeling. The goal is to understand whether different
recovery strategies lead to significantly different outcomes near a cutoff point.

## Problem Statement
Customers are assigned recovery strategies based on their expected recovery amount.
This project evaluates whether crossing a threshold (e.g., $1000) results in a measurable
change in actual recovery outcomes.
